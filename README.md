## ✨ Features

### 🎨 Design & Theme
- **Dark UI** with deep indigo, hot pink, cyan, and amber accent palette
- **Google Fonts** — Syne (headings) + DM Sans (body)
- **Noise texture overlay** for visual depth
- **Custom animated cursor** — glowing dot + trailing ring
- **Custom scrollbar** styled with gradient colors

### 🖥️ Sections
| Section | Description |
|---|---|
| **Loading Screen** | Full-screen loader with initials & animated gradient progress bar |
| **Navigation** | Frosted-glass fixed navbar with blur, slides in on load |
| **Hero** | Two-column layout with shimmer name animation + 3D floating card |
| **Skills** | Animated skill bars + floating tech chip cloud |
| **Projects** | Responsive grid cards with hover gradient sweep |
| **Experience** | Vertical timeline with gradient border |
| **Education** | Degree cards + animated language badges |
| **Contact** | Contact cards with pulsing ripple avatar |

### 🎞️ Animations
- ✅ Loading screen fade-out
- ✅ Custom glowing cursor with lag ring
- ✅ Live particle network canvas (background)
- ✅ Hero name shimmer gradient
- ✅ 3D floating card continuous rotation
- ✅ Scroll reveal — fade up / slide left / slide right (IntersectionObserver)
- ✅ Skill bars animate on scroll into view + shimmer sweep
- ✅ Infinite marquee for institutions
- ✅ Project cards top-border gradient sweep on hover
- ✅ Pulsing triple ripple rings on contact avatar

## 🚀 Getting Started

### View locally
Just open the file in any browser:
```bash
# Option 1 — double-click the file
open index.html

# Option 2 — VS Code Live Server
# Right-click index.html → Open with Live Server
```

### Deploy to GitHub Pages
```bash
git init
git add .
git commit -m "Initial commit: portfolio website"
git remote add origin https://github.com/Mohamedwaelfaro/my-portfolio.git
git branch -M main
git push -u origin main
```
Then go to **Settings → Pages → Source → main → Save** and your site will be live at:
```
https://Mohamedwaelfaro.github.io/my-portfolio
```
